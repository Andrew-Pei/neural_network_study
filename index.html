感知机与神经网络

从简单到复杂，理解AI的基石

感知机 (Perceptron)

最早的人工神经元模型，由 Frank Rosenblatt 在 1957 年提出。

局限性：只能解决线性可分问题，无法解决异或（XOR）问题

神经网络 (NN)

由多个神经元分层连接而成的网络，能够拟合任意复杂的非线性函数。

核心改进：非线性激活函数 + 隐藏层 + 反向传播

梯度下降

让神经网络学会"下山"的优化算法

什么是梯度下降？

梯度下降是一种优化算法，目标是通过迭代调整参数，使损失函数最小化。 想象你在山上，想下到最低点，你需要不断判断哪个方向是下坡方向。

三种梯度下降变体

批量梯度下降 (BGD)

使用全部数据计算梯度，稳定但慢

随机梯度下降 (SGD)

每次用一个样本更新，快但不稳定

小批量梯度下降 (Mini-Batch GD)

折中方案，最常用

03

误差反向传播

神经网络训练的核心算法

正向传播

输入 → 隐藏层 → 输出层 → 计算损失

反向传播

计算梯度 → 链式法则 → 更新参数

核心思想

反向传播算法本质是链式法则的应用。从输出层开始， 计算损失函数对各层权重和偏置的梯度，然后从后向前逐层传播。

∂L/∂W = ∂L/∂y × ∂y/∂z × ∂z/∂W

卷积神经网络 (CNN)

专门处理图像等网格结构数据的神经网络

卷积层

提取局部特征，保持空间结构

滑动窗口 + 点积运算
池化层

降维、减少计算量、增强平移不变性

最大池化 / 平均池化
全连接层

将特征映射到样本标记空间

加权求和 + 激活函数

05

神经网络三大函数

理解神经网络的核心组件

决策函数

将模型输出转化为最终决策

二分类: Sign(z)
多分类: Argmax(Softmax(z))

激活函数

引入非线性，使网络能够拟合复杂函数

Sigmoid、Tanh、ReLU

损失函数

衡量预测与真实标签之间的差距

MSE: 回归任务
Cross-Entropy: 分类任务

动量优化方法

利用历史梯度信息加速收敛

方法 特点 适用场景
SGD 仅依赖当前梯度，容易震荡 简单任务
SGD + Momentum 累加历史梯度，加速、平滑 一般任务
Adam 每个参数有独立学习率 大规模数据/参数
AdamW 权重衰减分离，泛化更好 Transformer (BERT/GPT)
